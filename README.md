# Server Monitoring Platform

A Flutter-based mobile monitoring application that displays real-time system metrics from a Windows PC using Prometheus and Grafana.

## Features
- Monitor PC CPU usage in real time
- Monitor Memory (RAM) usage
- Monitor Disk usage
- Monitor Network traffic
- Fetch metrics from Prometheus APIs
- View monitoring dashboards on Android devices
- Real-time system performance visualization

## Tech Stack
- Flutter
- Prometheus
- Grafana
- Windows Exporter
- Docker
- REST APIs
